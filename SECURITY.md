# Security Policy / 安全策略

## Reporting a Vulnerability

If you discover a security issue in BeeUrEi, please report it **privately** by email to
**<beeurei@163.com>**. **Do not open a public issue.**

We will acknowledge your report and work with you on a fix before any public disclosure.
Because BeeUrEi is an assistive tool for blind and low-vision users, we take privacy and
safety-critical reports especially seriously — particularly anything touching call privacy,
obstacle-avoidance discipline, or navigation gating.

When possible, please include:

- a description of the issue and its impact;
- steps to reproduce (a minimal proof of concept is ideal);
- the affected component (iOS app, `Packages/BeeUrEiCore`, or the self-hosted `server/`).

---

## 漏洞报告

如果你在 BeeUrEi 中发现安全问题，请通过邮件**私下**报告至 **<beeurei@163.com>**，
**请勿公开提交 issue。**

我们会确认你的报告，并在任何公开披露前与你一起修复。BeeUrEi 是面向视障与低视力用户的辅助工具，
我们对隐私与安全攸关的报告尤为重视——特别是涉及通话隐私、避障纪律或导航门控的问题。

报告时如能提供以下信息将很有帮助：

- 问题描述及其影响；
- 复现步骤（最好附最小可复现示例）；
- 受影响的组件（iOS App、`Packages/BeeUrEiCore`，或自托管的 `server/`）。
